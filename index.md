# Инструкция (Руководство) по работе с GIT

## 1. Что такое GIT

GIT - это система контроля версий (СКВ).

СКВ — это система, записывающая изменения в файл или набор файлов в течение времени и позволяющая вернуться позже к
 определённой версии.
